# Магазин теннисных ракеток

Проект интернет-магазина теннисных ракеток на Next.js.

## Функциональность

- Главная страница с популярными товарами
- Каталог всех ракеток
- Детальная страница товара с характеристиками
- Адаптивный дизайн

## Запуск проекта

1. Установите зависимости:

```bash
npm install
# или
yarn install
```

2. Запустите сервер разработки:

```bash
npm run dev
# или
yarn dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Особенности

- Использован Next.js App Router
- Изображения загружаются с `http://localhost:4000` (убедитесь, что сервер с картинками запущен, если они не отображаются)
- Статическая генерация страниц (SSG) для первых трех товаров
